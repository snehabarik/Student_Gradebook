A simple Python console-based student gradebook system that allows adding students, recording grades, and calculating averages.
# Student Gradebook System

## Description
This is a simple Python console-based gradebook system that allows users to:

- Add students
- Add grades for students
- View student grades
- Calculate average grades
- View all students with their averages

## Features
- Uses dictionary to store student data
- Input validation for grades
- Menu-driven interface
- Calculates averages automatically

## How to Run

1. Make sure Python is installed.
2. Run the file
